# React 앱 만들기

1. 터미널 열기
2. 프로젝트 준비

```sh
mkdir ReactProjects
cd ReactProjects
```

3. Vite를 사용한 React 프로젝트 생성

```sh
npm create vite@latest my-react-app -- --template react
```

4. 프로젝트 설정

```sh
cd my-react-app    # 새로 생성된 프로젝트 폴더로 이동
npm install        # 필요한 의존성 패키지 설치
npm run dev        # 개발 서버 실행
```
